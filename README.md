# todo-lab

Сервис для ведения списка задач с личными аккаунтами. Каждый пользователь видит только свои задачи.

## Возможности

- Регистрация и вход пользователей
- Создание, просмотр, редактирование и удаление задач
- Полная изоляция данных между пользователями
- Современный интерфейс на чистом JavaScript

## Технологии

- **Бэкенд:** Laravel (PHP), MySQL, REST API
- **Фронтенд:** Чистый HTML, CSS, JavaScript (ES6+)
- **Аутентификация:** JWT-токены
- **Развёртывание:** Docker, инструкции для ручной установки

## Для кого этот проект

- Для тех, кому нужен личный менеджер задач вне зависимости от сторонних сервисов
- Для разработчиков, которые хотят изучить пример полностью рабочего веб-приложения — от базы данных до пользовательского интерфейса

## Архитектура

Проект построен вокруг отдельного API, что позволяет при необходимости разработать дополнительные клиенты. Весь код открыт и доступен для изучения, модификации и самостоятельного развёртывания.
